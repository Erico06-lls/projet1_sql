# ⚽ Projet 02 - Machine Learning : Prédiction de matchs de football

## 📌 Description

Construction d'un modèle de Machine Learning capable de prédire le vainqueur d'un match de football à partir de données historiques internationales.

Le projet couvre les principales étapes d'un pipeline Data Science :
- 📊 Analyse des données
- 🛠️ Préparation et création de features
- 🤖 Entraînement d'un modèle
- 📈 Évaluation et prédictions

---

## 🗂️ Dataset

Source : International Football Results Dataset

- +49 000 matchs internationaux
- Période étudiée : **1994 - 2026**
- Informations utilisées :
  - équipes
  - scores
  - compétition
  - terrain neutre

---

## 🧠 Modèle utilisé

**Random Forest Classifier** 🌳

Features principales :
- forme récente des équipes
- différence de points
- différence de buts marqués/encaissés
- contexte du match

---

## 📊 Résultats

Accuracy obtenue :

**≈ 66%**

Le modèle permet également :
- 🔮 de prédire un match personnalisé
- 🏆 de simuler un tournoi complet
